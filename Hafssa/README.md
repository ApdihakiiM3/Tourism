# Hafsa Romantic Website

A modern, romantic single-page website created for Hafsa.

## Features
- Responsive design (mobile-first)
- Soft romantic color palette with gradients
- Glassmorphism card design
- Floating hearts animation
- Smooth transitions and hover effects
- Heartbeat animation on button
- Background music with mute/unmute
- Loading animation
- Click animations

## Files
- `index.html` - Main HTML structure
- `styles.css` - CSS styling and animations
- `script.js` - JavaScript for interactions

## Setup
1. Replace placeholder images in `index.html` with your actual images:
   - Change `https://via.placeholder.com/300x400/FFB6C1/FFFFFF?text=Hafsa` to your image URLs or local paths

2. Add background music:
   - Place a romantic MP3 file named `romantic-music.mp3` in the same directory
   - Or update the `src` in the `<audio>` tag in `index.html`

3. Open `index.html` in a web browser to view the website

## Customization
- Colors: Edit the gradients in `styles.css`
- Fonts: Already using Google Fonts (Poppins and Playfair Display)
- Animations: Modify timings and effects in `styles.css` and `script.js`

## Notes
- Music autoplay may be blocked by some browsers; users can click the mute button to enable
- Images are currently placeholders; replace with actual photos
- Fully responsive and optimized for mobile devices
